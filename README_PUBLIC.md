## English Quick Start

### 1. Download and Install

1. Open [Latest Release](https://github.com/WeAreTheArtMakers/modaitrader/releases/latest).
2. Download the installer for your OS:
   - macOS (`.dmg`)
   - Windows (`.exe`)
3. Install and launch `modAI Trader`.

#### macOS note
If Gatekeeper blocks startup:
- Right click the app -> `Open`
- Or open `System Settings -> Privacy & Security -> Open Anyway`

### 2. Activate License

1. Open the app and copy your `device_id` from the license activation screen.
2. Send **$500 USD equivalent** to one of the wallets below.
3. Email `admin@wearetheartmakers.com` with:
   - `device_id`
   - transaction hash (TXID)
   - network/coin used
4. Receive your device-bound license key.
5. Paste the key in the app and click **Activate License**.

#### Supported payment wallets
- `BTC`: `bc1p5qcj6lquhmpy4lz5qzkv4g3th6zh9keh5xdye5mt09k2fzmtd0ysucwhzs`
- `ETH`: `0x748bcF72eFA19eFEf100C5ED382D2c99520f057C`
- `SOL`: `3X4w9TJSjiQVsQrwhTZ2a7CEKNvgt7QRp15ZtmAd72nj`
- `BNB`: `0x7aAA6Bf8D0D33D7d26201E0Ad350f2be821f70f1`
- `MONAD`: `0x748bcF72eFA19eFEf100C5ED382D2c99520f057C`

### 3. Connect Binance API

1. Create API keys:
   - Testnet: [testnet.binancefuture.com](https://testnet.binancefuture.com)
   - Mainnet: [binance.com/api-management](https://www.binance.com/en/my/settings/api-management)
2. In app, open **Bot Control**.
3. Choose mode: `Testnet` (recommended first) or `Mainnet`.
4. Enter API Key + Secret and save.

### 4. Start First Strategy

1. Select a strategy/template.
2. Pick your symbol (example: `BTC/USDT`).
3. Start bot and monitor:
   - Dashboard (PnL, positions)
   - Risk panel
   - AI signals

### 5. Recommended First-Day Workflow

1. Use **Testnet** only.
2. Run at least 10-20 paper trades.
3. Keep risk score controlled.
4. Move to Mainnet only after stable test results.

## FAQ (EN + TR)

### EN

**Q: Is the license subscription-based?**  
A: No. Current model is one-time payment (**$500**) with lifetime validity for one device.

**Q: Can I test before live trading?**  
A: Yes. Use Binance Testnet first.

**Q: What API permissions are required?**  
A: Enable `Reading` and `Futures`. Do **not** enable withdrawals.

### TR

**S: Lisans aylik abonelik mi?**  
C: Hayir. Guncel model tek odeme (**$500**) ve tek cihaz icin omur boyu gecerliliktir.

**S: Canliya gecmeden once test edebilir miyim?**  
C: Evet. Once Binance Testnet kullan.

**S: Hangi API izinleri gerekli?**  
C: `Reading` ve `Futures` acik olmali. Cekim (`withdraw`) izni acilmamali.
