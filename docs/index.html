Safe usage note

Start on Testnet before Mainnet.

  • Use Testnet for first setup and strategy validation.
  • Keep API withdraw permission disabled.
  • Use hard budget caps and direction lock during early runs.
  • Review risk panel and open positions before increasing size.
  • Use Chat Control for plan-first flow: AI analyzes, user approves, then execution starts.
  • In Chat Control, review confidence/risk output and use one-click debug when an action is blocked.

FAQ

Critical answers before onboarding

What are the current license plans?

Trial 5 Days (Free), Monthly ($250), Yearly ($2700), and One-Time Lifetime ($5000).

Can I test before payment and upgrade later?

Yes. Start with Trial 5 Days, then upgrade the same Device ID to Monthly, Yearly, or Lifetime.

When does full access open?

Full modules open only after a valid device-bound license key is activated.

Which platforms are supported?

Current public release includes macOS (Intel + Apple Silicon), Windows x64, and Linux ARM64 (Raspberry Pi 5).
